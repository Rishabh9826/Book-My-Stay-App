# Book My Stay App – Use Case 1

## Overview
This project demonstrates the basic entry point of a Java application using a simple hotel booking system scenario. It focuses on how a Java program starts execution and produces console output.

## Features
- Displays welcome message
- Shows application name and version
- Demonstrates program start and termination

## Technologies Used
- Java (Core Java)

## How to Run

### Step 1: Compile
javac UseCase1HotelBookingApp.java

### Step 2: Run
java UseCase1HotelBookingApp

## Output
=====================================
   Welcome to Book My Stay App
=====================================
Application Name : Hotel Booking System
Version          : v1.0

Thank you for using the application!
Application terminated successfully.

## Concepts Covered
- Class
- main() method
- static keyword
- Console output
- String literals
- Linear execution flow
